# Decisao 0001: Arquitetura Inicial

## Contexto

O sistema sera usado para gerar conteudo editorial com apoio de IA, partindo de backlog, artigos antigos e estilo autoral.

## Decisao

Criar um repositorio separado para o Grimorio Content Engine, com estrutura para contexto, grimorio, agente, backlog, amostras de escrita, templates, rascunhos, prontos, publicados e metricas.

## Consequencias

- O sistema pode evoluir sem contaminar o nucleo de ShadeVanSoul.
- Artigos antigos podem ser importados como referencia de estilo.
- O agente editorial tera contexto explicito.
- A automacao de publicacao fica adiada ate que o fluxo editorial esteja maduro.

## Status

Accepted.
