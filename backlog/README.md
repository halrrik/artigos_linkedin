# Backlog Editorial

Este backlog guarda ideias e oportunidades de conteudo.

Use os arquivos conforme o tipo de material:

- `inbox.md`: ideias brutas ainda nao triadas.
- `themes.md`: temas recorrentes.
- `series.md`: possiveis sequencias de conteudo.
- `articles.md`: artigos longos.
- `linkedin-posts.md`: posts curtos ou medios.
- `responses.md`: respostas a debates, comentarios e posts de terceiros.

## Status sugeridos

- idea
- triaged
- drafting
- reviewing
- approved
- scheduled
- published
- archived
- blocked
