# Operador de Publicacao

## Estado inicial

Este agente ainda nao deve publicar automaticamente.

## Papel futuro

Quando o sistema estiver maduro, este operador podera:

- listar conteudos prontos;
- sugerir agenda;
- preparar publicacao em ferramenta externa;
- alertar sobre posts pendentes;
- registrar publicacoes realizadas;
- coletar metricas basicas;
- identificar comentarios que merecem resposta.

## Regra de seguranca editorial

Nenhum conteudo deve ser publicado automaticamente sem uma decisao explicita de aprovacao.

## Status permitido

- ready_to_review
- approved_to_schedule
- scheduled
- published
- needs_revision
- blocked
